IoT: MQTT Inference System
Deskripsi Proyek

Sistem backend lengkap yang dibangun menggunakan Go untuk menerima data dari sensor melalui protokol MQTT. Data yang masuk kemudian diinferensikan secara real-time oleh model Machine Learning yang berjalan di Python. Hasilnya disimpan di MongoDB dan divisualisasikan secara langsung di frontend menggunakan WebSocket.

Fitur Utama:
  • Integrasi MQTT Broker untuk data streaming.
  • Komunikasi antar-service (Go & Python) melalui gRPC.
  • Penyimpanan data NoSQL dengan MongoDB.
  • Dashboard real-time dengan WebSocket.
Teknologi:
GoPythonMQTTgRPCMongoDBWebSocket
